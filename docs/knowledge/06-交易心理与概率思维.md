# 06 - 交易心理与概率思维 (Mark Douglas 内化)

> 老板 2026-07-01 指示: 学习路径 P0, "Mark Douglas 概率思维内化".
> 底书: Mark Douglas《交易心理分析》(Trading in the Zone, 2000) + 《自律的交易者》(The Disciplined Trader, 1990).
> 对照错误: 06-29 创业板 4.145 浅止损割日内低点 / 芯片第二刀 1.545 偏早 / 盘中恐慌卖踏空 ~224 元.
> 核心命题: **大多数人亏钱不是因为技术不行, 是因为用"确定性思维"做概率游戏.**

## 一、核心范式转换: 确定性思维 → 概率思维

| 维度 | 确定性思维 (亏钱心态) | 概率思维 (Douglas 的 zone) |
|---|---|---|
| 单笔结果 | "这笔一定要赚" / "这笔会亏" | 这笔是分布中的一个样本, 不知道结果 |
| 亏损 | 情绪威胁, 痛, 要避免 | 预先接受的成本, 做生意的费用 |
| 急杀 | "要崩了快跑" | 一个样本, 对照结构位, 没破=噪音 |
| 预测 | 努力猜对方向 | 不需要猜对, 优势长期赢就行 |
| 自我 | "我对了/错了" | 与对错无关, 只看是否按系统执行 |
| 样本 | 单笔即评判 | 攒 ≥30 笔按 setup 判期望 |

**悖论**: 技术分析教你"预测" (这是会发生), Douglas 教你"放下预测" (我不知道会发生什么, 但我知道长期赢). 大多数人卡在"想预测正确"的自我满足里, 一旦预测落空就情绪化操作. 06-29 恐慌卖 = 典型: 看见急杀, 大脑预测"要崩", 情绪逃离, 实际是分布中的正常噪音, 下午 V 回.

## 二、Douglas 五个基本事实 (概率思维的基石)

1. **任何事情都可能发生** (Anything can happen).
   即使你有最强优势, 下一根 K 线任何结果都可能出现. 一根大资金单、一条新闻、一个政策, 瞬间改变. 06-29 早盘急杀没人提前知道会 V 回.

2. **赚钱不需要知道下一步会发生什么** (You don't need to know what happens next).
   优势是统计性的, 你靠"赢的次数×赢的幅度 > 输的次数×输的幅度"赚钱, 不靠"这笔猜对". 赌场不预测下一手牌, 靠概率优势.

3. **每个优势的输赢序列是随机的** (Win/loss sequence is random).
   假设你的 setup 胜率 55%, 你**无法知道**接下来 10 笔哪 5-6 笔赢、哪 4-5 笔输. 连亏 5 笔后仍可能第 6 笔输. 任何"该反弹了"的想法 = 赌徒谬误.

4. **优势只表明一件事比另一件事更可能发生** (An edge = one outcome more likely than another).
   不是"会赢", 是"赢的概率略高". 55% 胜率 = 接近抛硬币, 区别只在长期. 单笔看, 几乎是 50/50.

5. **市场每一刻都是独特的** (Every moment is unique).
   每个 setup 是独立样本, 上一笔的结果不传染这一笔. "上次这里亏了所以这次不敢" / "上次这里赚了所以这次重仓" = 都是把独立样本当连续.

## 三、Douglas 七个一致性原则 (持续盈利者的行为)

1. **我客观地确认优势** — setup 有量化定义 (VCP/Wyckoff/Turtle 突破), 不是"感觉".
2. **我预先定义每笔风险** — 入场前写止损价 + 仓位 (rules.yaml + plan_stop_loss), 不临场改.
3. **我完全接受风险, 否则不交易** — 入场前想清楚"这笔亏 X 元我能接受吗", 不能接受就缩仓位或不做.
4. **我按优势行动, 毫不犹豫** — 信号触发即执行 (突破即买/止损即卖), 不等"再看看".
5. **市场给多少我收多少, 不追** — 到目标位落袋, 没到持有, 不因贪婪加码.
6. **我持续监控犯错的可能** — 盘中每次想操作前问: "这是系统信号还是情绪反应?"
7. **我明白这些原则绝不违反** — 不是"尽量遵守", 是"违反=亏钱, 没有例外".

## 四、06-29 恐慌卖案例逐条剖析

**事件**: 06-29 早盘创业板 ETF 急杀, 在 4.145 (日内低点附近) 浅止损割 1000 股, 下午 V 回到 4.364 (+3.02%), 踏空 ~0.22 元/股.

| 违反的原则/事实 | 06-29 的具体表现 | 正确做法 |
|---|---|---|
| 事实1 (任何事可能发生) | 看见急杀 = 预测"要崩" | 急杀是其中一个可能, V 回也是其中一个可能 |
| 事实3 (输赢序列随机) | 把"这笔亏损"当威胁而非样本 | 这是分布中的一个亏损样本, 接受 |
| 原则2 (预定义风险) | 创业板止损 <2% (数学 bug) | 用 ATR 结构止损 4.04, 不用浅止损 |
| 原则3 (完全接受风险) | 没接受 ETF 波动, 急杀恐慌逃离 | 入场前接受"这笔可能到 4.04", 到了才动 |
| 原则4 (按优势行动) | 急杀时是**情绪**驱动卖, 不是信号 | 跌破 4.04 **且收阴**才是信号, 急杀不破=持有 |
| 原则6 (监控犯错) | 没问"这是信号还是恐慌" | 卖前问一句, 答案是恐慌=不卖 |

**根因一句话**: 用确定性思维 ("急杀=要崩, 快跑") 做概率游戏, 没接受单笔是样本, 把情绪当信号.

## 五、内化操作: 盘前/盘中/盘后的概率思维触发器

> 不是"读一遍就懂", 是"每次操作前过一遍". 像盘前 checklist 一样机械执行.

### 盘前 (09:25, 配合 `pre_market_checklist.md` 第5节)
- [ ] 默认今天: **任何持仓可能亏损到止损价, 这是概率成本, 不是威胁**.
- [ ] 每持仓想: "这笔亏到止损 (具体价) 我接受吗?" 不接受 → 缩仓位或不持有.
- [ ] 提醒自己: 今天发生的任何单笔结果, **不评判对错**, 单样本无意义.

### 盘中 (急杀/急拉时)
- 急杀 → 想"这是分布中的一个样本, 不是确定性崩盘". 对照结构位: 没破 = 噪音持有, 破了且收阴 = 信号减仓.
- 急拉 → 想"这是其中一个可能结果". 到目标位落袋, 没到持有.
- **每次想操作前, 强制问一句: "这是 setup 信号还是情绪反应?"** 答不上来"是信号" → 不操作.
- 早盘 09:30-10:00 情绪极端区, 默认不操作 ([F] 铁律).

### 盘后 (15:10, 配合复盘)
- 单笔盈亏**不评判**: 不写"今天又错了/对了", 写"这是 setup X 的第 N 个样本, 结果 +/−Y".
- 攒够 ≥30 个同 setup 样本 (`backtest_hypothesis`), 才判这个 setup 的期望 (expectancy).
- 连亏不慌、连赢不飘 — 都是随机序列的局部.

| Douglas 原则 | 项目落地 |

| 原则1 客观确认优势 | scorer 量化 (VCP/Wyckoff/Turtle/moneyflow), setup 打标签 (`log --setup`) |

## 七、一句话内化

> **"我不知道这笔会赚会亏, 但我知道按系统做 ≥30 笔会赚. 所以这笔我按系统做, 无论结果."**
