# 盘前 Checklist (每日 09:25 前过一遍)

> 基于 01-04 已学铁律 + 06-29 首战教训产出. 目的: 防盘中执行变形.
> 06-29 错误: 创业板浅止损割日内低点 / 芯片第二刀偏早 / 瞎猜东财成本.
> 盘前过一遍 = 把"知道"变成"做到". 漏一项 = 当日不操作.

## 1. 账户与成本 (铁律: 报数前必查db)
- [ ] 每持仓跑 `log cost <code>` — 真实成本, 不瞎猜
- [ ] 跑 `risk_metrics` — 组合HEAT / 板块集中度 / 最大单仓
- [ ] 跑 `market_regime` — 派发日数 / FTD / 风险等级 (NORMAL/CAUTION/HIGH/SEVERE)
- [ ] 跑 `sentiment` — 情绪温度 (<40谨慎, ≥40中性)

## 2. 市场结构 → 今日基调
- [ ] NORMAL → 可进攻 (开新仓OK)
- [ ] CAUTION + 情绪<40 → 防御 (不开新仓, 只减)
- [ ] HIGH/SEVERE → 防御 (只减不加, 考虑清仓)
- [ ] 查 close_scan 昨日 industry_flow — 资金流出板块不碰 ([J]假设)

## 2.5 今日操作裁决 (集合竞价后 09:25, 一句话定调)
> 综合 §2 市场结构 + 情绪 + 板块资金流 + edge_scanner候选质量, CFO 给**一句话裁决**。
> **必须给牌, 老板必须听。** 07-02教训: 560780接飞刀, 因盘前没拦死。

- 🟢 **绿牌 操作**: regime=NORMAL + 情绪≥40 + 持仓板块**不在**资金流出榜 + edge_scanner有干净候选 → 可开**体系内**新仓
- 🟡 **黄牌 仅动体系内**: regime=CAUTION 或 情绪30-40 或 持仓板块轻度流出 → 只执行已计划体系单, **不加仓破位/不接刀**
- 🔴 **红牌 今天不操作**: regime=HIGH/SEVERE 或 情绪<30 或 组合上日破-3% 或 持仓板块**多日**资金流出([J]/[L]) → **只止损/减仓, 绝不开新仓, 绝不接跌**

> **07-02 回判**: 弱市(组合-1.81%起) + 硬科技多日流出([J] 07-01半导体-260亿) + 组合续破-3% + 体系涨市数据未验跌市 → **红牌**。买560780(-8%接刀)属违反红牌。
> **[L]铁律**: 板块-8%先查**多日**资金流 — 多日流出+破位=出货(别碰), 单日巨量V收=恐慌出逃(可买)。

## 3. 持仓触发位定形
- [ ] 每持仓: 支撑/阻力/回踩位具体价格 (写下来)
- [ ] 止损位: ETF 用 ATR (宽基 ≥4%, 行业 ETF ≥5%), **禁 <2% 浅止损** ([D]铁律)
- [ ] 个股用结构位 (跌破前日低**且收阴**才动)
- [ ] 强势板块留底仓 ≥30% ([E]铁律)

## 4. 今日计划 (写 `log plan`)
- [ ] 加仓候选 + 触发价 + 仓位% (凯利 sizer 算)
- [ ] 减仓计划 (减在急拉, 不减在回踩)
- [ ] 入场框架分清: 强势票形态确认 / 弱势票等支撑 ([A]假设)

## 5. 心理准备 (Mark Douglas 概率思维, 详 docs/knowledge/06)
- [ ] 默认今天: **任何持仓可能亏损到止损价 = 概率成本, 不是威胁**
- [ ] 每持仓想: "亏到止损 (具体价) 我接受吗?" 不接受 → 缩仓或不持有
- [ ] 提醒: 今天任何单笔结果**不评判对错**, 单样本无意义 (≥30 笔才判 setup 期望)
- [ ] 纪律优先: 盘中每次想操作, 强制问"这是 setup 信号还是情绪反应?" 答不上"信号"→不操作

## 6. 禁止区 (铁律)

- [ ] ❌ 日内低点卖 (等10:00后 + 破结构位确认)
- [ ] ❌ 浅止损 <2% ([D]数学bug)

## 7. 盘中执行口诀
- 早盘急杀 → 等 10:00 后, 看是否破结构位, 没破 = 噪音持有
- 急拉 → 减仓点 (强势留底仓≥30%)
- 回踩缩量不破 → 加仓点 / 持有
- 放量破位 → 真破位 (缩量破位=洗盘)
