## 二、验证主回路 (技能如何"落地→验证→固化")

技能不落地 = 白学. 每学一个技能走闭环:

```
学技能 → 提炼可测假设 (标 [A]-[Z]) → 写入 daily.md 验证区
→ 满 5 工作日 → backtest_hypothesis 回测 (n≥30 才充分)
→ 命中率高 → 固化 rules.yaml / scorer 权重
→ 命中率低 → 修正或删除假设
```

关键纪律 (已在 CLAUDE.md / daily.md):
- **单日印证 ≠ 正式回测** (单日是观察样本, 不是统计判定)
- **不提前固化** (满 5 工作日才判定)
- 区分"先用着后验证"(直观铁律) vs "待验证"(新假设)
