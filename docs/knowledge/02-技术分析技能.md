# 技术分析技能

> 供理财顾问CFO盘面判断用。理论出处: Steve Nison《日本蜡烛图技术》、John Murphy《金融市场技术分析》、Joseph Granville 均线八法则、量价关系经典理论。案例数据均经本项目 `data/ohlcv/*.parquet` 核对(截至2026-06-26收盘)。

## 一、均线系统与格兰维尔八法则

**MA周期实战含义**: MA5=周线(短线情绪)、MA10=半月(短波段)、MA20=月线(中线生命线)、MA60=季线(A股中线分水岭)。MA60约一个季度, 是机构建仓成本线, **站上MA60=中线多头格局, 破MA60=中线转弱**。再往上有MA120半年线、MA250年线(牛熊分界)。

**格兰维尔八法则(4买4卖)**, 把均线当"趋势轨道+乖离回归":
- 4个买点: ①均线走平转升、价突破MA(趋势启动) ②价回踩MA不破再上(**回踩确认**, 最稳买点) ③价跌破MA但MA仍上升且快速拉回(假跌破/洗盘) ④价大幅跌破MA、乖离过大(超卖反弹)
- 4个卖点: ⑤均线走平转跌、价跌破MA(趋势转弱) ⑥价反弹到MA不过再下(反弹受阻) ⑦价突破MA但MA仍下行且快速回落(**假突破/多头陷阱**) ⑧价大幅升破MA、乖离过大(超买回落)

**金叉/死叉**: 短期均线上穿长期=金叉看多(MA5×MA10短线、MA10×MA20波段、MA20×MA60中线); 下穿=死叉。

**多头排列**: 价>MA5>MA10>MA20>MA60, 全部向上=最强多头。**空头排列**: 价<MA5<MA10<MA20<MA60=最弱空头。
- 多头排列范例: 159801芯片ETF close1.547 > MA5 1.49 > MA10 1.39 > MA20 1.29 > MA60 1.12, 教科书多头, 中线最强。
- 空头排列范例: 515650消费50ETF close0.955 < MA5 0.96 < MA10 0.98 < MA20 1.01 < MA60 1.06, 全空头, 观望。

**乖离率BIAS=(价−MA)/MA**, 过大必回归均值。恒瑞06-26 close48.67 vs MA60 52.23, BIAS=−6.8%(价在MA60下方), 配合缩量属超卖待回归区。

## 二、量价关系

**核心: 量在价先。成交量是价格的燃料。** 项目 `technical_scorer.py` 已加载 volume 列却完全未用于评分, 这是当前最大缺口。

- **放量突破**: 突破关键位+量比≥1.5~2倍=有效突破, 主力进场。东财06-22 18.45→20.97, 量1103万 vs 06-17的274万=**4倍量突破**, 教科书级可信突破。
- **缩量上涨(价升量缩背离)**: 价涨量不增=上涨乏力, 警惕。健康多头需"量价齐升"。
- **量价齐升**: 价涨量增, 最健康多头趋势。
- **天量见天价**: 历史天量+高位=见顶, 主力出货; **地量见地价**: 极致缩量+低位=见底。
- **破位放量 vs 缩量(关键判别)**: 破位**放量**=主力出逃真破位; 破位**缩量**=洗盘假破位, 常是黄金坑。**恒瑞06-26破位50.46→48.67, 量比0.90x(缩量!)** → 更像洗盘而非真破位, 这从量价层面印证了"破位反弹"判断的合理性。若那天是2倍量破位, 结论相反。
- **创业板缩量**: 159915 06-26量比1.07x平量, 上行中量能不济, 后续不补量则上行乏力。

**用成交量验证突破**: 突破日量比≥1.5 + 后续2~3日缩量回踩不破突破点=真突破(东财模式); 突破日缩量或次日放量跌破=假突破。

## 三、K线形态与破位反弹

- **V型反转**: 恐慌急跌后强势资金介入快速拉回, 形成V, 常因利空出尽/主力抄底。恒瑞06-26破位48.67后06-29冲52.97(实时盘面)=V反转, 涨幅8.8%。V反转需底部放量配合方有效。
- **突破回踩**: 突破阻力后回踩该位(原阻力变支撑)确认再上=格兰维尔第二买点。东财06-22破20.97后回踩20整数关+MA20(19.04)/MA60(19.57)整理, close20.07站上=典型突破回踩确认。
- **假突破(多头陷阱)**: 突破阻力但量不配合, 快速回落跌破=诱多(格兰维尔第七卖点)。识别: 突破日缩量+次日阴线吞没+跌破突破点。
- **锤头线(长下影)**: 下影≥实体2倍、上影极短, 出现在下跌末端=见底反转, 下影越长多头反击越强。
- **破位反弹三浪结构**: 破位(A浪下跌)→反弹(B浪)→再探底/再上(C浪)。恒瑞: 06-26破位48.67(A)→06-29反弹冲52.97(B), 若回踩MA60(52.23)确认再上55(C)=三浪反弹。**回踩确认是加仓点而非卖点**——这与06-29"盘中急杀卖在低点"的教训直接相关。

## 四、支撑压力位识别

- **前高前低**: 近期高点=压力, 低点=支撑。东财06-23 high21.88成压力, 21.57→21.17逐次下移=下降通道上轨。
- **整数关**: 20/50/100等心理位。东财20整数关=支撑, 恒瑞50整数关=压力转支撑。
- **缺口**: 跳空缺口=强支撑/压力。突破缺口(放量)不回补=强势; 衰竭缺口(高位)易回补=见顶。
- **密集成交区**: 成交量集中价格带=强支撑/压力。
- **压力位决定止盈**: 多头冲压力位量能不足=止盈; 放量突破压力=持有/加仓。恒瑞目标55=前期压力延伸+整数关, 冲55若缩量则止盈。

## 五、趋势判断

- **下降通道**: 高点低点同步下移。东财06-23 high21.88→06-25 21.57→06-26 21.17, 高点逐降=下降通道上轨。需突破21.88并站稳才转势。
- **上升通道**: 高点低点同步上移, 连接低点画上升趋势线(支撑), 破线=转势。
- **独立强势行情**: 大盘跌它涨/抗跌, 均线多头排列, 板块逻辑独立。159516近20日+52% vs 上证−4.9%=独立强势(卡脖子材料+装备链)。识别: 相对强度RS极高、抗跌、逻辑独立于大盘。但+52%后乖离大, 回踩1.85(rules.yaml已设提醒)=回踩买点, 符合格兰维尔第二买点。

## 七、项目落地建议

`scorers/technical_scorer.py` 当前只判断多头/空头排列+MACD+RSI, 缺口大。建议增强(优先级排序):

1. **量价验证(最高优先, 当前完全没用volume)**: 在 `score()` 加 `vol_ratio = last_vol / mean(vol[-20:])`。
   - 放量突破(量比≥1.5 且 价破前高/破MA20)→ +10
   - 缩量上涨(价升 量比<0.7)→ −8(背离)
   - 缩量破位(价跌 量比<0.8)→ +5(洗盘信号, 反向)
   - 放量破位(量比≥1.5 且 价跌)→ −10(真破位)

2. **MA60中线信号**: 当前只比排列, 缺独立事件。
   - 价站上MA60 且 MA60近5日走平转升 → +10(中线转多)
   - 价跌破MA60 → −10(中线转弱)

3. **金叉死叉**: 加 `ma10=_sma(closes,10)`, 存前一日的ma5/ma10做交叉检测。
   - MA5上穿MA10/MA20(金叉)→ +8; 下穿(死叉)→ −8

4. **乖离率BIAS**: `bias=(price-ma20)/ma20`。>15%超买−5; <−15%超卖+5。

5. **回踩确认信号**: 破位后N日内缩量回踩MA不破再上→ +8(格兰维尔第二买点)。

6. **monitor加规则(rules.yaml)**: 放量大阳破前高提醒、缩量破位(假破位洗盘)提醒、长下影锤头线提醒、MA60破位/收复提醒。

代码骨架(加在 `technical_scorer.py` 的 `score()` 内, volume已可用):
```python
vols = [r.get("Volume") or r.get("volume") for r in rows if r.get("Volume") or r.get("volume")]
vol_ratio = vols[-1] / (sum(vols[-20:]) / 20) if len(vols) >= 20 else 1.0
ma10 = _sma(closes, 10)
# 量价
prev_close = closes[-2]
if price > closes[-20:].max() and vol_ratio >= 1.5:  # 放量破前高
    s += 10; detail["vol_breakout"] = "放量突破"
elif price > prev_close and vol_ratio < 0.7:  # 缩量上涨背离
    s -= 8; detail["vol_divergence"] = "价升量缩"
# MA60中线
if price > ma60 > 0 and ma60 > _sma(closes[:-1], 60):
    s += 10; detail["ma60"] = "中线多头"
elif ma60 > 0 and price < ma60:
    s -= 10; detail["ma60"] = "中线转弱"
```

**最高ROI**: 先做#1量价验证——volume已在手里、零额外数据成本, 却能直接判别"恒瑞缩量破位=洗盘"这类关键场景, 正好补06-29教训的根因。
