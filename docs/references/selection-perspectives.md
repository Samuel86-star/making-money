## 1. 低 ROE + 弹性拐点 (我们最缺的维度)

**核心**: 不追高ROE白马 (已price in), 找**ROE低但即将释放业绩**的票。
- 标的处于研发投入期 (收入未兑现, ROE被压低)
- 即将进入收获期 (产能/产品落地, ROE弹性拐点向上)
- 市场未充分定价 (估值低, 赔率高)

**为什么有用**: 我们 scorer 的 fundamental_scorer 偏"好公司" (高ROE/稳增长), 缺"拐点公司"视角。黑马多从低ROE拐点出, 不是白马。

**A股例子**:
- 半导体材料/设备: 研发期长, 国产替代放量前ROE低, 一旦验证拐点, 业绩弹性巨大 (159516逻辑)
- 创新药: BD出海+管线兑现前ROE低, 验证后跳升 (600276创新药逻辑)

**怎么补我们缺口** (待5日验证后考虑):
- fundamental_scorer 加因子: `roe_inflection` = ROE当前低(<8%) + 研发投入高(R&D/营收>5%) + 营收拐点(环比加速)
- 与现有"好公司"因子互补, 不替换

## 2. 供给端创造需求 (科技通胀叙事)

**核心**: AI/算力/半导体的需求是**供给端创造**的 (技术突破造出新需求), 不是传统经济周期需求拉动。
- 传统需求: 收入↑ → 消费↑ (顺周期, 经济驱动)
- 供给创造需求: 技术突破 → 新产品(算力/模型) → 催生新需求 → 科技型通胀

**为什么有用**: 解释AI主线为何能脱离经济周期持续强势。我们前面困惑"PMI弱为何科技涨", 这条给答案 — 科技是供给驱动, 跟PMI弱不矛盾。

**A股例子**:
- 光通信/算力: AI模型突破 → 算力需求爆发 → 光模块/服务器涨价 (供给创造)
- 存储芯片: AI终端放量 → HBM/DRAM需求 → 涨价周期

**怎么用**:
- 选股时优先"供给创造需求"赛道 (科技通胀受益), 而非纯顺周期
- 解释昨天 (06-30) 创业板+3%/通信ETF+4.7% — 科技主线是这逻辑, 不是PMI驱动

## 3. 中国全球比较优势 (国产替代+壁垒)

**核心**: 只选中国有**全球比较优势或卡脖子国产替代**逻辑的环节, 不选无壁垒的同质化竞争。
- 比较优势: 全球份额领先 + 成本/技术壁垒 (光伏/电池/光通信)
- 国产替代: 卡脖子环节 + 国产化率提升空间大 (半导体设备/材料)

**为什么有用**: 给选股加**全球视角**, 我们现有 scorer 只看公司层面, 缺产业链定位。卡脖子环节=政策+资金+订单三重催化。

**A股例子**:
- 半导体设备 (159516): 国产化率<20%, 卡脖子, 政策扶持
- 光通信: 中国全球份额领先, 比较优势明确
- 创新药: BD出海 = 全球需求, 不只靠国内

**怎么用**:
- 选股时加一道筛: "这票所在环节中国有没有全球优势/国产替代逻辑?" 没有 → 降级
- morning_scan 候选可加此维度过滤
