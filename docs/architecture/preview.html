技术栈

全栈 Python 3.12+, 零外部中间件, 纯本地运行

维度选择原因
语言Python 3.12+数据源SDK/量化库生态
存储SQLite × 5 + JSON × 4单用户, 无需 PG
数据源东财 push2 / 腾讯 / 同花顺 / 新浪免费/稳定/批量
通知macOS osascript本地单机, 无需钉钉/微信
调度cron + scheduler.py简单可靠, 带 WAL 持久化
限流三闸门 RateLimiter防封 (抄 a-share-quant)
防重入threading.Lock 非阻塞LLM/评分超时防堆积
