## 诚实裁决

### ✅ Turtle sys1 (20日突破) — 真edge, 可信
- 10日 horizon: **edge +1.00%, 胜率51.8% (base 46.9%, +4.9pp)**, 中位+0.34%
- 唯一胜率明显高+中位为正的信号 → 不靠少数大赢撑, 分布性正期望
- **结论**: morning_scan 给 sys1 加 +5 合理, 实战可倚. 扣交易成本(~0.3%)后 10日 edge 仍有 ~+0.7%.

### ⚠️ Turtle sys2 (55日突破) — 长线趋势型, 低胜率高方差
- 20日 edge +0.85% 但胜率42.1% < base, 中位-2.78% → "少数大赢撑均值" 典型趋势跟踪
- **结论**: 有效但高方差, 需配宽止损+长持有期. 不适合短线/小资金高频.

### ✅ Wyckoff 派发 — 有效**回避**信号 (负edge方向对)
- 5日 edge -0.16% → 派发后确实**弱于base** = 出货警告方向正确
- **结论**: scorer 给派发 -10 合理, 作**不买过滤器**用. 不是做空信号 (A股融券难).

### ❌ VCP (Minervini) — **无edge**, 需修正或弃用
- 三horizon edge 全 ≈ 0 (-0.09%/+0.04%/-0.03%), 胜率45.6% < base, 中位-0.73%~-3.60%
- **根因诊断**: 当前 `_detect_vcp` 检测"收缩形态"但**不含突破确认** → 在很多"收缩后继续震荡/下跌"的假形态上也命中. 真 VCP 应在**收缩+放量突破前高当日**才触发.
- **结论**: VCP 强化 [A] 假设的量化基础**塌了**. 需二选一:
  1. 修正 detector: 加突破确认 (价破末段高 + 放量) 再触发, 重跑回测验 edge
  2. 暂从 scorer 摘除 VCP 加分 (+12/+6), 等 [A] 假设07-07日记回测再判
- **现状**: VCP 加分仍在中 (本次只验证不改, 避免连续未验改动). 标记待修正.

### ✅✅ VCP 修正后 (2026-07-01 当日修正 + 突破确认, 重测)
加"当日突破right段高 + 放量1.3×"确认后, 重跑回测:
- 命中从 12137 → **4353** (2.8× 更严选, 剔除假形态)
- edge 从 ≈0 → **三horizon全正**: 5日+0.17% / 10日+0.34% / 20日+0.23%
- 胜率仍 <base (46.2% vs 47.8%), 中位负 → 突破型特征: 少数大赢撑均值, **需紧止损骑赢家**, 不是高胜率信号
- **裁决**: 修正成功, VCP 从"无edge"变"modest正edge". 弱于 Turtle sys1 (+1.00%) 但真实. scorer 的 +12/+6 保留合理.
- 验证回路闭环: 回测发现 → 修正 → 重测确认. 这是不验=自欺的反面教材.

## 关键教训 (验证的价值)

1. **不验 = 自欺**. VCP 我刚建时信心满满 (强化[A]), 回测打平 base. 没这步会一直以为它是 edge.
2. **base rate 是正的** (市场涨) → 所有信号在涨市看着都还行. 真考验是跌市. 当前结果只在涨市有效.
3. **中位 << 均值** → 偏态分布. Turtle sys2/VCP 中位负但均值正 = 少数大赢. 实战单笔风险高, 仓位不能大.
4. **Turtle sys1 是当前最可靠信号**. morning_scan 的 +5 加分有据.

## 止损建模对比 (2026-07-01 加, 关键发现)

加 `realized_return_with_stop`: 任意日 low ≤ entry×(1-stop%) → 止损出场. 重测三模式:

| 信号 (10d) | 无止损 edge | 3%止损 edge | 5%止损 edge | 无止损胜率 | 5%止损胜率 |
|---|---|---|---|---|---|
| Turtle sys1 | +1.00% | +0.47% | **+0.75%** | 51.8% | 42.7% |
| VCP (20d) | +0.23% | +0.33% | **+0.60%** | 41.0% | 22.3% |
| Wyckoff吸筹 | +0.19% | +0.23% | +0.25% | 47.5% | 38.6% |
| Turtle sys2 | +0.51% | +0.17% | +0.31% | 44.2% | 28.8% |

**三大发现**:

1. **3%止损 = edge杀手**. Turtle sys1 10d +1.00%→+0.47% (砍半), 所有信号胜率崩到14-39%, median全-3%. A股日均振幅2-3.5%, 3%被反复打脸 → **实证项目铁律"ETF止损<2%是数学bug"** (连3%都太紧). 这是用数据印证了之前只在文档里的规则.

2. **5%止损是甜区**. Turtle sys1 10d 恢复到 +0.75% (无止损的75%), 印证铁律"宽基≥4%/行业ETF≥5%". 5%止损能: 砍掉灾难尾 (median -5% 封顶) + 放过日内噪音 + 保留趋势.

3. **VCP 是真正的突破型信号** — 5%止损下 20d edge 从 +0.23% 翻倍到 **+0.60%**. 突破策略本质: 小亏多次 (median -5%封顶) + 少数大赢 (均值+1.38%). 没止损 = 让失败交易烂到尾; 有5%止损 = 切断左尾. 这证明 VCP 修正后不只是"有正edge", 而是"按突破策略设计正确运作".

**实战指导**:
- 突破型信号 (VCP/Turtle) **必须配 ≥5% 止损**, 否则左尾吃光edge.
- Turtle sys1 是唯一在所有止损 regime 下都正edge 的信号 = 最稳健.
- 3%及以下止损在A股 = 自残 (数据证实, 非理论).

**局限**: 止损按 stop_price 成交 (保守, 假设无滑点/缺口). 实战缺口/滑点会让真实止损略差. 未建模次日开盘买入 (用close[T]=信号日收盘).
